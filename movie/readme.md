## Goal
Exploratory Data Analysis of movie dataset to understand distributions, outliers, and correlations.

## Techniques Used
- Descriptive statistics
- Outlier detection (IQR)
- Correlation analysis
- Data visualization (matplotlib, seaborn)
- Normality testing (Q-Q plot)

## Dataset
movies.csv (IMDb + financial data)

## Key Insights
- Budget and revenue are highly skewed
- Strong correlation between budget and revenue
- Several extreme outliers in revenue and budget
