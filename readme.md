# Data Analytics Portfolio

This repository contains a collection of data analytics projects focused on exploratory data analysis, statistical testing, and business insights using Python.

The goal of this portfolio is to demonstrate practical skills in:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical hypothesis testing
- Data visualization
- Business-oriented analytics and decision making

---

## 📂 Projects

### 1. 📊 Statistical Analysis (Students Performance Dataset)
Analysis of student exam performance.

**Techniques used:**
- One-sample t-test
- Two-sample t-test
- Paired t-test
- Proportion z-test

**Key focus:**
Understanding statistical differences between groups and interpreting p-values.

---

### 2. 🎬 Movie Dataset Analysis (EDA)
Exploratory analysis of movie industry data.

**Techniques used:**
- Descriptive statistics
- Outlier detection (IQR method)
- Correlation analysis
- Distribution analysis
- Q-Q plot for normality check
- Data visualization (Matplotlib, Seaborn)

**Key insights:**
- Relationship between budget and revenue
- Distribution of ratings
- Presence of outliers in financial data

---

### 3. 📞 Customer Churn Analysis (EDA)
Analysis of telecom customer churn behavior.

**Techniques used:**
- Conditional probability analysis
- Lift calculation
- Segment-based churn analysis
- Feature impact evaluation

**Key insights:**
- High churn in month-to-month contracts
- Fiber optic users show higher churn probability
- Payment method impacts churn risk

---

### 4. 💰 Customer Value & Business Decision Analysis (CLV)
Business-focused customer value analysis.

**Techniques used:**
- Expected value modeling (EV)
- Customer segmentation (risk/value)
- Budget optimization
- Scenario analysis

**Key insights:**
- Identifying high-value customers
- Optimizing marketing budget allocation
- Evaluating campaign profitability

---

### 5. 🏠 Housing Data Analysis (Regression)
Analysis of California housing prices.

**Techniques used:**
- Linear regression
- Feature engineering
- Standardization
- Multicollinearity check (VIF)
- Residual analysis
- Log transformation

**Key insights:**
- Strong influence of location features
- Non-linear relationship with target variable
- Improved model stability after transformation
